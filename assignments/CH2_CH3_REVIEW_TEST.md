## Introduction: The Thinker Block Method
Before you begin Part 2, remember that a **Thinker Block** is a visual tool used to represent an unknown quantity—what mathematicians call a **variable**. Instead of using a letter like "x," we use a block to represent a value we don't know yet. 

*   If two people have the same amount of something, they each have one identical block.
*   If one person has "5 more" than another, they have one block plus the number 5.
*   By using these blocks, you can "see" the math and solve complex problems by balancing the totals.
